소방관·구급대원의 안타까운 현실

출동 중 길을 막는 차량 하나가 골든타임 5분을 송두리째 앗아갑니다. 소방차뿐 아니라 구급차도 같은 문제에 직면합니다. 목숨을 걸고 달리면서도 "왜 안 비켜주냐"는 좌절감은 직업 소진의 큰 원인이 됩니다.

경로 차단 → 골든타임 초과 → 소방관·구급대원 번아웃
02 — How It Works

GPS 기반
3단계 실시간 매칭

불필요한 차량에게 알림을 보내지 않습니다. 진행 방향 전방에 있는 차량만 정확하게 타겟팅합니다.

Step 01 — 송신

실시간 GPS 브로드캐스트

출동 소방차·구급차의 GPS 좌표 및 이동 방향 데이터를 0.5초 주기로 서버에 송신합니다.

⚡ 0.5초 갱신
→
Step 02 — 필터링

반경 스마트 타겟팅

진행 방향 전방 1~2km 반경 안에 있는 일반 차량만 타겟팅합니다. 관계없는 차량은 제외됩니다.

📡 2km 반경 필터
→
Step 03 — 연동

내비게이션 즉시 알림

티맵, 카플레이 등 운전자가 보고 있는 화면에 슬림 상단 바 형태로 즉시 표출됩니다.

📱 내비 직접 연동
03 — UI / UX

슬림 상단 바 구조
