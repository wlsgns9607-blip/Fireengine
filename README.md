# 🚒🚑 골든패스 (Golden Pass)

> **소방차 · 구급차 실시간 경로 연동 포트폴리오 프로젝트**  
> 소방차·구급차의 GPS 위치를 운전자 내비게이션에 실시간 연동하여 골든타임을 지키는 플랫폼

---

### 👨‍💻 기획자 소개 및 프로젝트 의의

**"화면 설계를 넘어 대기업 인프라 연동과 시스템 아키텍처를 설계하는 서비스 기획자, 조진훈입니다."**

본 프로젝트 '골든패스(Golden Pass)'는 단순한 모바일 화면 UI/UX 기획에 머무르지 않고, 대기업(현대자동차 등)의 V2X 커넥티드 카 인프라 및 실시간 GPS 데이터 매핑 로직을 전제로 설계된 정통 서비스 기획(Service/System Planning) 마스터피스입니다.

단순한 기능 명세를 넘어 비즈니스의 거대한 판을 짜고, 탑티어 개발자 및 디자이너와의 거대 규모 협업을 리드할 수 있는 다재다능한 기획 역량을 본 프로젝트를 통해 증명합니다.

*(※ 저작권법에 의해 본 기획의 지적 재산권은 원작자에게 귀속되어 있으며 푸터 명세를 통해 보호받고 있습니다.)*

---

## 📌 프로젝트 개요

골든패스는 긴급차량 출동 시 발생하는 두 가지 문제를 동시에 해결합니다.

- **시민 운전자**: 사이렌 방향을 인지하지 못해 판단이 늦어지는 문제
- **소방관·구급대원**: 경로 차단으로 인한 골든타임 초과 및 번아웃

## 🏗️ 페이지 구성

### 01 Hero
- 브랜드 태그라인 및 핵심 지표 3가지 노출
  - 골든타임 **5분**, GPS 갱신 **0.5초**, 알림 반경 **2km**
- 배경: 레드·골드 블러 오브 애니메이션 (`pulse` keyframe)

### 02 Problem
- 두 가지 고통을 나란히 대비 (`problem-grid` 2열 레이아웃)
  - 🚗 시민 운전자: 방향 불명 → 판단 지연 → 경로 차단
  - 🚒 소방관: 경로 차단 → 골든타임 초과 → 번아웃

### 03 Mechanism
- GPS 기반 3단계 매칭 프로세스
  1. **송신** — 0.5초 주기 GPS 브로드캐스트
  2. **필터링** — 진행 방향 전방 1~2km 스마트 타겟팅
  3. **연동** — 티맵·카플레이 등 내비게이션 즉시 알림

### 04 UI Demo
- 차량 내비게이션 목업 (실시간 시뮬레이션)
- 스크롤 진입 시 7초 루프 애니메이션 자동 실행
  - 소방차 이동 + 파동 링 → 슬림 알림 바 드롭다운 → 토스트 팝업

### 05 Rewards
- 시민 운전자: 안전점수 + 뱃지 + 커피 쿠폰
- 소방관·구급대원: 명예 뱃지 + 국민 감사 메시지

### 06 Closing
- 브랜드 철학 마무리

## ⚙️ JavaScript 동작

### 스크롤 reveal

```js
// .reveal 클래스 요소가 뷰포트(10%)에 들어오면 .visible 추가
const revealObserver = new IntersectionObserver(callback, { threshold: 0.1 });
```

### 내비 데모 애니메이션 (`startNaviAnimation`)

| Phase | 시점 | 동작 |
|---|---|---|
| 1 | 즉시 | 소방차 이동, 파동링, 반경 원 시작 |
| 2 | 0.8s | 알림 바 드롭다운 |
| 3 | 1.4s | 빨간 점 깜빡 + 화살표 흐름 |
| 4 | 3.5s | "영웅의 길" 토스트 팝업 |
| 5 | 7s | 전체 리셋 후 루프 재시작 |

`#uidemo` 섹션이 뷰포트 25% 이상 노출될 때 트리거됩니다.
